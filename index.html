Fitness Class Booking Website

I have documented and designed a website utilizing a data warehouse, featuring a fitness class booking system. The site serves two user groups: customers and instructors. Customers can create an account, log in, and manage their bookings by making or canceling them. Instructors have their own section to add and manage classes. The website is built using HTML, CSS, JavaScript, PHP, and MySQL
